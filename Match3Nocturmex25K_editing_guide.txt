QUICKSTART FOR NEWBIES
================================================================================

How to apply these edits in DaVinci Resolve or Final Cut Pro:

SLOW MOTION:
  - Resolve: Right-click clip → Retime Controls → Speed → 50% (or 25%)
  - FCP: Select clip → Retime menu → Slow → 50%

ZOOM (Push-in):
  - Resolve: Inspector → Transform → Zoom → Add keyframes (1.0 → 1.2)
  - FCP: Select clip → Transform → Add keyframes to Scale

SOUND EFFECTS (SFX):
  - Layer SFX on separate audio track above natural sound
  - Reduce natural sound by -3 to -6 dB during SFX (audio ducking)
  - Common SFX: impact boom, whoosh, tension drone

COLOR GRADING:
  - Resolve: Color page → Apply LUT or adjust Contrast/Saturation
  - FCP: Color Board → Increase contrast, warmth for dominance
  - 'Punchy' = +10-20% contrast, slight saturation boost
